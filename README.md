## Configuration

The system is configured via `config.json`:

```json
{
  "focusMode": {
    "enabled": true,
    "type": "Advanced Technology Professional",
    "locked": true,
    "unlockKeyword": "unlock"
  }
}
```

## Usage

### Focus Lock Status
By default, the system operates in **LOCKED** mode as a Pro Expert Advanced Technology Programming Professional Developer.

### Unlocking Focus Mode
To unlock the focus mode, the user must explicitly state the unlock keyword exactly:

```
unlock
```

**Important**: The keyword must be spelled exactly as shown. Variations like "Unlock", "UNLOCK", or "un-lock" will NOT work.
